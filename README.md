# Enterprise Sales Lakehouse

## Overview

An end-to-end Azure Data Engineering project that demonstrates a production-style Medallion Architecture (Bronze, Silver, Gold) using Azure Data Factory, Azure Databricks, Delta Lake, Azure Data Lake Storage Gen2, Azure SQL Database, and Power BI.

## Tech Stack

- Azure Data Factory
- Azure Data Lake Storage Gen2
- Azure Databricks
- PySpark
- Delta Lake
- Azure SQL Database
- Power BI
- Git & GitHub
